──────────────────────────────────────────────────────────────────────────────
WHAT LANDED (self/codegen.hexa — inspection-verifiable, byte-eq by construction)
──────────────────────────────────────────────────────────────────────────────
The escape→stack non-escape analyzer previously discovered ONLY top-level `let`
bindings of a fn body. This sub-task extends it to discover `let` bindings
declared INSIDE nested if / else / while / for / match block bodies, proving the
binding's lifetime is bounded by its enclosing block.

  * _stack_child_bodies(node)  — enumerate a node's nested block bodies
    (then_body / else_body / body / each match arm.body).
  * _stack_collect_flat_lets(stmts, out)  — recursively gather flat-struct
    `let p = Pt{…}` candidates at ANY nesting depth.
  * _stack_collect_arr_lets(stmts, out)   — recursively gather bounded-array
    `let a = [e0…eN]` candidates at ANY nesting depth.
  * _stack_noescape_scan / _stack_noescape_arr_scan — rewritten to scan the
    recursively-collected candidates over the WHOLE fn body using the EXISTING
    whole-body escape proof (_stmt_escapes_name / _stmt_escapes_arr_name, which
    already recurse into nested bodies, so an escape ANYWHERE is caught).
  * LetStmt arm (gen2_node) — UNCHANGED. It already fires for nested LetStmts
    because gen2_node recurses into block bodies; it emits the SAME
    gen2_stack_alloc_flat / gen2_stack_alloc_array_lit strings. The stack object
    therefore lands in EXACTLY the enclosing C block scope — the binding never
    outlives its descriptor (no dangling).

Soundness of nesting: a nested binding whose lifetime crosses its block (stored
to an outer-scope var, returned, captured, pushed to an outer container) is
flagged by the whole-body escape scan (the bare-Ident / call-arg / store / push
rules) and stays on the heap. Name shadowing is conservatively safe: the escape
scan runs over the whole body for the name, so if the SAME name escapes anywhere
(an outer alias), the nested candidate is rejected too — we never wrongly stack-
allocate an escaping binding. GATED on HEXA_STACK_ALLOC (default OFF) → byte-
identical emit when OFF. FN-scoped, reset per fn.
