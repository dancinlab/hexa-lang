# Atkin-Lehner involution count for Γ₀(N) — closed form

Theorem (Atkin–Lehner, 1970):
  | { Atkin-Lehner involutions of Γ₀(N) } | = 2^ω(N)
  where ω(N) = number of distinct prime divisors of N.

The Atkin-Lehner involutions are indexed by exact divisors d ∥ N
(i.e. d | N with gcd(d, N/d) = 1). For N = ∏ p_i^{a_i} an exact
divisor is any product of a subset of the prime powers p_i^{a_i},
which gives exactly 2^ω(N) choices.

# Anchor — ω(N) parity via μ(N)  (squarefree N only)

For squarefree N (i.e. μ(N) ∈ {-1, +1}), the Möbius function
gives  μ(N) = (-1)^ω(N), so the PARITY of ω is g5-verifiable via
the `mu` calculator path. Captured verdicts:

  mu_N1.txt  : μ(1)  =  1  (🔵)  →  ω(1)  even (=0)
  mu_N2.txt  : μ(2)  = -1  (🔵)  →  ω(2)  odd  (=1)
  mu_N6.txt  : μ(6)  =  1  (🔵)  →  ω(6)  even (=2)
  mu_N30.txt : μ(30) = -1  (🔵)  →  ω(30) odd  (=3)

These four parities are consistent with the ω(N) column above.

For N = 12 (NOT squarefree, μ(12) = 0) this anchor does not apply;
ω(12) = 2 is read from the prime-power decomposition 12 = 2²·3.
