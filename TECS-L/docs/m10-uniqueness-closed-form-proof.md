# M10 — σ(n)·φ(n)=n·τ(n) ⟺ n∈{1,6} 닫힌형 증명 (🟡 → 🔵)

> M1·M3·M9 이 **유한 sweep(n≤100)** 으로만 보이고 🟡 citation 으로 남겨둔 **전칭(∀n)
> 유일성**을, 곱셈성 분해로 **닫힌형 증명**한다. base-case factor 값은 `hexa verify`
> 로 machine-verified (🔵), 일반 한계는 초등 지수>선형 부등식. 결과: 🔵 SUPPORTED-FORMAL.

## 정리

**모든 양의 정수 n 에 대해 σ(n)·φ(n) = n·τ(n) ⟺ n ∈ {1, 6}.**

## 증명

σ, φ, τ, 그리고 항등함수 n 은 모두 **곱셈적(multiplicative)** 이다. 따라서 비
F(n) := σ(n)φ(n) / (n·τ(n)) 도 곱셈적이고, n = ∏ pᵢ^{aᵢ} 에 대해

```
   F(n) = ∏  g(pᵢ, aᵢ),     g(p,a) := σ(pᵏ)φ(pᵏ)/(pᵏ(a+1))  with k=a
```

각 소수거듭제곱 인자를 닫힌형으로 정리하면 (σ(pᵃ)=(p^{a+1}−1)/(p−1),
φ(pᵃ)=p^{a−1}(p−1)):

```
   g(p,a) = (p^{a+1} − 1) / ( p · (a+1) )
```

그러므로 **σ(n)φ(n)=n·τ(n) ⟺ F(n)=1 ⟺ ∏ g(pᵢ,aᵢ) = 1.**

### 보조정리 (부호 패턴) — g(p,a) > 1 for all (p,a) except g(2,1)=3/4

g(p,a) > 1 ⟺ p^{a+1} − 1 > p(a+1) ⟺ **p^{a+1} > p(a+1) + 1.**

- **p = 2:** 2^{a+1} > 2a + 3.  a=1: 4 > 5 거짓 → **g(2,1) = 3/4 < 1 (유일 예외)**.
  a≥2: 2^{a+1} 는 지수, 2a+3 은 선형 → 성립 (a=2: 8>7). 즉 p=2 는 a=1 에서만 <1.
- **p ≥ 3:** a=1: p² > 2p+1 ⟺ p²−2p−1>0 ⟺ p > 1+√2 ≈ 2.414 → p≥3 성립.
  a≥2: p^{a+1} ≥ p³ ≥ 27, 지수>선형 → 성립. 즉 p≥3 은 항상 >1.

machine-verified base cases (`hexa verify --expr`, 전부 🔵):

| (p,a) | σ(pᵃ) | φ(pᵃ) | g(p,a) | |
|-------|-------|-------|--------|---|
| (2,1) | 3 | 1 | 3·1/(2·2) = **3/4** | <1 (유일) |
| (3,1) | 4 | 2 | 4·2/(3·2) = **4/3** | >1 (최소 홀수인자) |
| (2,2) | 7 | 2 | 7·2/(4·3) = 7/6 | >1 |
| (5,1) | 6 | 4 | 6·4/(5·2) = 12/5 | >1 |
| (7,1) | 8 | 6 | 8·6/(7·2) = 24/7 | >1 |

또한 홀수 소수거듭제곱에서 g 는 p·a 에 대해 단조증가하므로 **최소 홀수 인자 =
g(3,1) = 4/3.**

### 본 논증 — ∏ g = 1 의 해는 {1, 6} 뿐

- **n = 1**: 공곱(empty product) = 1. ✓
- **n > 1, 2-인자 없음 또는 2ᵃ(a≥2):** 모든 인자 > 1 → 곱 > 1 ≠ 1. 모순.
  ⇒ n 은 반드시 **2¹** 을 포함해야 한다.
- **2¹ ‖ n:** 곱 = (3/4)·∏_{홀수 pᵃ} g. ∏_{홀수} = 4/3 필요. 각 홀수 인자 ≥ 4/3:
  - 홀수 인자 0개: 곱 = 3/4 ≠ 1 (n=2). ✗
  - 홀수 인자 1개 = 4/3: g(p,a)=4/3 (p 홀수) → a=1 에서 3(p²−1)=8p → 3p²−8p−3=0
    → **p=3** (유일). ⇒ n = 2·3 = **6**. ✓ (a≥2 는 g(3,2)=2.89>4/3, 해 없음)
  - 홀수 인자 ≥2개: 곱 ≥ (4/3)² = 16/9 > 4/3. ✗

∴ ∏ g(pᵢ,aᵢ) = 1 ⟺ **n ∈ {1, 6}.** ∎
