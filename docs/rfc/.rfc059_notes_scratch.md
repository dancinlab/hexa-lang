# RFC 059 backward — investigation notes (P1② design)

## ag_tape reverse-mode (stdlib/flame/ag_tape.hexa, 1508 lines)
- Tape = ONE farr (node v3). header[4]: [0]=op count, [1]=legacy chain link,
  [2]=reg_tids farr, [3]=reg_gids farr. Per-op = 22 slots:
  kind(0) · 8 ids(1..8) · 4 dims(9..12) · 6 pgrad(13..18) · 3 igrad(19..21).
  ids stored as floats via to_float/to_int.
- RECORD: each ag_<op> wrapper calls verified nn_lib fwd, allocs+saves bwd
  state, _ag_push appends a node. NOT reimplementing vjp math — reuses nn_lib.
- REPLAY: ag_backward_reg(tape,out_tid,seed) walks ops latest→first; per-op
  output-grad looked up by output tid in a PER-TENSOR GRAD REGISTRY (Decision 3,
  _ag_reg_grad zero-allocs on first touch, _ag_reg_acc += accumulates). Each op's
  bwd accumulates input/param grads into registry keyed by tid → handles linear
  chains, attn fan-in, param reuse, and RESIDUAL FAN-OUT uniformly.
- MULTI-OBJECTIVE already present: ag_seed_grad(tape,tid,seed) pre-populates a
  registry slot WITHOUT walking — caller seeds head_g's grad, then
  ag_backward_reg seeds head_a + walks once; both contribute. (live in
  flame_anima_multi_objective_test.hexa main loop).
- 13 op kinds: rmsnorm·linear·rope·lmhead·swiglu·embed·attn·add·rope_mh·slice·
  silu_gate·rmsnorm_mh·attn_dt. lmhead bwd = nn_lm_head_bwd (L655 nn_lib).

## 4 V3 ops → math + required grads
- head_g: second tied-style LM head, separate W_g. logits_g[k]=Σ_j W_g[k·d+j]·z[j].
  ∂L/∂z = Σ_k dl_g[k]·W_g[k], ∂L/∂W_g = dl_g ⊗ z. = existing nn_lm_head_bwd.
- cross (L_psi): c=cos(logits_a,logits_g)=dot/(‖a‖‖b‖); L=c²/4. dL/dc=c/2.
  d/da = (c/2)(b/(‖a‖‖b‖) − c·a/‖a‖²); symmetric for b. (multi-obj test L167).
- purefield (PureFieldFFN): out = E_A(x) − E_G(x), E=Linear→GELU→Linear.
  tension=mean(out²). bwd = two linear-GELU-linear vjp chains; d_out splits +1
  to E_A, −1 to E_G. GELU' = Φ(x)+x·φ(x).
- tension_proj (L_route): tension_l = mean(out_l²) per layer; L_route=λ·Σ_l t_l².
  ∂L_route/∂out_l = λ·2·t_l·(2/N)·out_l = (4λ t_l/N)·out_l → seeds purefield bwd.

## full-position head (nn_lib L808/L834, PR #1481)
- nn_lm_head_fwd_allpos(temb,Z,logits_out,T,V,d): logits[t·V+k]=Σ_j temb[k·d+j]·Z[t·d+j]
- nn_ce_loss_allpos(logits,targets,T,V): (1/T)Σ_t −ln softmax(logits[t])[targets[t]]
- backward needed: ∂L/∂logits[t·V+k] = (1/T)(softmax(logits[t])[k] − [k==targets[t]])
  ∂L/∂Z[t] = Σ_k dl[t,k]·temb[k];  ∂L/∂temb += Σ_t dl[t] ⊗ Z[t]  (tied accumulate)
  = per-row nn_lm_head_bwd accumulating over T. NOT YET wired into ag_tape.

## existing backward template
- ag_k_lmhead (ag_tape L1070): og=reg_grad(logits); nn_lm_head_bwd→dtemb,dzT;
  reg_acc(temb,dtemb), reg_acc(zT,dzT). This is the per-op pattern to clone for
  an allpos lmhead node (loop nn_lm_head_bwd over T rows into shared temb).
- ag_k_linear (L963): matmul_bwd_auto dW=xᵀ·dy, dx=dy·Wᵀ (byte-eq nn_linear_bwd).

## fp-tol convention
- RFC 040 §2.2 TOL_MATMUL class (cited FLAME.tape L98, PERF.md L258). fwd cross-impl
  = fp-tol; pure-scalar bwd = byte-eq. RFC 059 Path-A draft uses central-diff vs
  analytic 1e-7 (F-RFC059-C1/C3). multi-obj test grad-check ~1e-4..1e-6.
- grad-check = finite-diff vs analytic, NOT byte-eq (reverse-mode accumulates fp).
